To install run chmod +x su.sh && ./su.sh. Then once booted, run apt install sudo, then sudo apt install nano, then sudo apt update.
To run Debian 10 again, do ./su.sh, and enter on the first question A, then n on the next 3 popups and it will start.
After start, run sudo apt update to clean storage.
